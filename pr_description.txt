🎯 **What:** The testing gap addressed
This PR addresses testing edge cases in `ContactController.SubmitContactForm`, verifying rate limiting attributes and ensuring HTTP methods are properly annotated.

- Valid request parsing and processing
- `BadRequest` returns for completely null requests
- `BadRequest` returns for invalid payloads with empty/whitespace properties

- Validation that the rate limiting attribute `[EnableRateLimiting("ContactLimiter")]` is applied to prevent abuse vectors.

✨ **Result:** The improvement in test coverage
The method's logic and declarative security attributes are comprehensively verified.
